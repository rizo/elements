# Elements

A collection of small modules, types and functions that will make your life easier.

Currently the following modules are provided:

```
Elements
 ├── Base
 ├── Control
 │  ├── Exn
 │  └── Fn
 ├── Data
 │  ├── Array
 │  ├── Char
 │  ├── Counter
 │  ├── Dict
 │  ├── Either
 │  ├── Float
 │  ├── Int
 │  ├── Iter
 │  ├── List
 │  ├── Map
 │  ├── Option
 │  ├── Persistent_vector
 │  ├── Range
 │  ├── Result
 │  ├── Stream
 │  ├── String
 │  └── Void
 └── System
    └── IO
```
